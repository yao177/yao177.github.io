亲密关系

08-28-2020 6 min read

今天听了一节播客：翻转电台Flip Radio——问答VOL8|来吧，你们最关心的的那个问题，亲密关系

链接：亲密关系
下面是一点notes

1 亲密关系是一种纯粹的关系

1.1 传统亲密关系

是一个共同体

1.2 现代亲密关系 自由恋爱

与家族、工作、国家、宗教无关
一种纯粹的关系

私人空间：物理，心理
唯一例外：两性亲密关系
特殊性

2 亲密关系重要吗

非常重要！！！
自我认可：亲密关系可以充分满足自我认可
自信心/基础安全感：宾我，私人亲密关系

2.1 为什么亲密关系是…的来源？

自信的参与：别的失败都有原因，亲密关系的失败是纯粹自我否定的失败->完全否定对方（从全面肯定出发，又必须说服自己；很容易得神经症；人格心理学原理：会变得和那个人特别像）
非制度：反抗制度
依靠药物、性等释放出来的不是自信，是原始本能

黑尾猴：一生中最重要的就是找到一个异性

心理动力学：基础人格
阿尼姆；阿尼姆斯
如果没有经历过健康的亲密关系，就会活成理想另一半的样子

2.2 为什么容易失败

  • 维系亲密关系的纽带缺失 => 亲密关系容易损坏
  • 社会流动性 -> 选择多 -> 边际收益越来越大
  • 失败对个人的否定是根本的失败 => 囚徒困境博弈 -> 先脱身的伤害小，后脱身的伤害大 => 纳什：不合作 -> 面对损失会过度理性，过度高估损失 => 敏感地轻易放弃

我们之间的关系还好不好？经不起琢磨、反思？反思本身就具有破坏力

开放关系减少了亲密关系的伤害

开放关系降低了风险，同时可以在每一段关系中全心投入 => 解决囚徒困境

开放关系真的好吗

阿尼姆和阿尼姆斯只有一个！
开放关系先否定了个人亲密关系的存在
个人主义时代 -> 个人主义需要不断排除其它因素，只留自我、无限纯化 => 例外：只有亲密关系是例外且唯一的 -> 任何其它原因都会让纯化大打折扣

2.3 亲密关系脆弱的四大原因

  1. 纯粹关系：构成外部必然性的基础消失；剩下每日无穷反思（感情好不好/和Ta在一起是否必要）；亲密关系无基础
  2. 亲密关系的选择过多
  3. 亲密关系的风险太大：成为囚徒困境博弈（每个人都想全身而退；博弈的均衡解决是不合作）
  4. 亲密关系的替代路径——开放关系在逻辑上与亲密关系背道而驰：不是一个好路径
  5. 今日的亲密关系是知识的重灾区：异化知识和陈词滥调对一个人的亲密关系影响很大

3 5种亲密关系的关键隐喻

陈词滥调 & 异化知识

3.1 “健康”的隐喻 ❌

将亲密关系比作疾病

健康与否
情感治疗

危害：只要去医院，必然能查出问题，没有人的身体时纯粹健康的 -> 亲密关系时肯定有问题的 -> 一再问亲密关系好不好，会找到诸多瑕疵 => 禁不起问！！！

治疗：将制度和规范放入感情 -> 设计技术性方法、流程（感情本身的非制度性被破坏）

3.2 “企业”的隐喻 ❌

将亲密关系比作企业

需要经营
看重结果：不要分开

危害：为了实现目的 => 马克思主义的合理化 -> 程序性和形式框架

治疗：停止！ => 否则亲密关系的重要性大打折扣

3.3 “镜像神经元”的隐喻 ❌

将亲密关系比作镜像

换位思考
充分了解对方
融合成一个人：你中有我，我中有你

危害：基于个人主义的例外规则 => 尼采的虚无主义批判 -> 当基督教定义的完全意义的善恶未降临时，人们会开始怀疑基督教本身 -> 当开始想着两个人能够完全坦率、完全理解、完全换位思考时，做不到！遂，开始对亲密关系本身产生怀疑、进行批判

3.4 “信仰”对隐喻 ❌

将亲密关系比作信仰

心灵鸡汤
心诚则灵：毫无杂念，心无偏私

危害：“信仰”VS“迷信” -> 心足够诚，便能灵验，感情会越变越好 => “信仰”不等于“心想事成”；信仰不关注现实 => 对感情有信仰的人，不是满足于“到死不分开”，而是会关注于细节也能好；无关信仰

3.5 “自我”的隐喻 ❌

将亲密关系比作自我

爱别人，首先肯定要爱自己
只有会爱自己，才会爱别人
不会爱自己，就不会爱别人

危害：纯粹的个人主义对亲密关系这个例外项也会有排除 => “爱”这个词，已经包含了自我否定，是比自己重要的存在 -> 宾我爱主我？宾我无限肯定主我 -> 宾我对主我的一切反思意义都不大，放弃反思，导致任性 -> 连宾我对自己的反思都不能接受，就更不能接受亲密关系对自我的阻碍了 => 不可能爱别人！

4 对亲密关系的建议

4.1 把握真正的关系

当今社会，缺乏真正关系，就没有客观真实感受。

4.2 如何把握亲密关系

先通过其它真正的朋友关系（好；本真；有摩擦），提升自己对关系的把握，提高承担人际风险的能力，敢于接受亲密关系

脆弱的亲密关系，如何抵抗脆弱性？（暂时不得而知……）
